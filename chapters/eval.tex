\section{การตรวจสอบความถูกต้องของแบบจำลอง (Model Validation)}
ผู้วิจัยได้ดำเนินการตรวจสอบความถูกต้องของฟังก์ชันวัตถุประสงค์ (QUBO Formulation) โดยใช้กรณีศึกษาพื้นฐานเพื่อยืนยันประสิทธิภาพของอัลกอริทึม ดังนี้:
\begin{itemize}
    \item \textbf{การทดสอบด้วยทฤษฎีเกมพื้นฐาน:} จากการทดลองใช้ปัญหา Prisoner's Dilemma พบว่าระบบสามารถระบุจุดดุลยภาพแนชที่สภาวะการยอมรับสารภาพทั้งสองฝ่าย (1,1) ได้อย่างถูกต้อง โดยให้ค่าพลังงานต่ำสุด (Energy) เท่ากับ 0.0 เมื่อรวมกับค่าบทลงโทษ (Penalty)
    \item \textbf{การเข้ารหัสความเสียใจ (Regret Encoding):} ระบบสามารถแปลงความสัมพันธ์ของผลตอบแทนในรูปแบบความพึงพอใจสูงสุดให้อยู่ในรูปสมการ QUBO ได้อย่างแม่นยำผ่านการใช้ Slack Variable ขนาด 5 บิต ซึ่งรองรับส่วนต่างของผลตอบแทนได้สูงสุดถึง 31 หน่วย
\end{itemize}

\section{การวิเคราะห์ผลการทดลอง (Result Analysis)}
จากการจำลองสถานการณ์บนโครงสร้างต้นไม้ (Cyber Tree) ที่ประกอบด้วยสินทรัพย์สารสนเทศ 18 รายการ ผู้วิจัยได้วิเคราะห์ผลลัพธ์เชิงกลยุทธ์ดังนี้:

\subsection{กรณีผู้เล่น 2 คน (1 Attacker vs 1 Defender)}
จากการทดสอบโดยกำหนดงบประมาณของผู้โจมตีที่ 75 หน่วย พบว่า:
\begin{itemize}
    \item \textbf{จุดสมดุลแนช:} ระบบตรวจพบดุลยภาพแนชที่สถานะ (DFS, Random) โดยมีผลตอบแทนของผู้เล่นเป็น (80, 115) และค่าพลังงานเท่ากับ 0.0
    \item \textbf{อิทธิพลของกลยุทธ์:} กลยุทธ์การสำรวจแบบ DFS มักให้ผลตอบแทนแก่ผู้โจมตีสูงกว่าในกรณีที่สินทรัพย์มูลค่าสูงถูกจัดวางไว้ในชั้นที่ลึกของต้นไม้
\end{itemize}

\subsection{กรณีผู้เล่น 3 คน (2 Attackers vs 1 Defender)}
ผู้วิจัยได้ขยายการทดลองสู่ระบบผู้เล่น N คน โดยกำหนดงบประมาณ A1=100 และ A2=50 พบว่า:
\begin{itemize}
    \item \textbf{ผลลัพธ์ดุลยภาพ:} พบจุดสมดุลแนชที่สถานะ (BFS, DFS, ASC) ซึ่งให้ผลตอบแทนรวมเป็น (99, 71, 25) และค่าพลังงานรวมเท่ากับ 0.0
    \item \textbf{การแข่งขันระหว่างผู้โจมตี:} เมื่อมีผู้โจมตีมากกว่าหนึ่งราย การเข้าถึงโหนดจะเป็นลักษณะลำดับการเข้าถึง (Turn-based) ซึ่งส่งผลให้จุดสมดุลเปลี่ยนไปตามงบประมาณที่แตกต่างกันของแต่ละฝ่าย
\end{itemize}

\section{การเปรียบเทียบและการประเมินประสิทธิภาพ (Comparison and Evaluation)}
\begin{itemize}
    \item \textbf{ประสิทธิภาพของอัลกอริทึมแบบ Hybrid:} การใช้แนวทาง Hybrid Quantum-Classical ช่วยให้สามารถแก้ปัญหาที่มีขนาดตารางขนาดใหญ่และมีจุดสมดุลหลายจุด (Multiple Equilibria) ได้สำเร็จโดยใช้จำนวนคิวบิตที่ลดลงจากการนำคิวบิตกลับมาใช้ใหม่ในฝั่งการประมวลผลแบบดั้งเดิม
    \item \textbf{ข้อจำกัดด้านความถูกต้อง:} ในปัญหาขนาดเล็กระบบทำงานได้แม่นยำสูง แต่ในปัญหาขนาดใหญ่คำตอบอาจไม่ใช้ดุลยภาพที่แท้จริงเสมอไปเนื่องจากลักษณะทางกายภาพของพลังงานที่อาจติดอยู่ในค่าต่ำสุดเฉพาะจุด ซึ่งจำเป็นต้องมีการทำซ้ำการทดลองเพื่อหาคำตอบที่ดีที่สุด
\end{itemize}

\section{ข้อเสนอแนะและสรุปผล (Suggestions and Summary)}
\begin{itemize}
    \item \textbf{การพัฒนาต่อยอด:} ในกรณีที่พบหลายจุดสมดุล ควรเพิ่มการวิเคราะห์เพื่อนบ้าน (Neighbor Analysis) เพื่อค้นหาจุดสมดุลที่มีประสิทธิภาพสูงสุดสำหรับระบบโดยรวม
    \item \textbf{สรุปภาพรวม:} แบบจำลองที่พัฒนาขึ้นพิสูจน์แล้วว่าสามารถระบุกลยุทธ์ที่เหมาะสมที่สุดสำหรับทั้งฝ่ายโจมตีและป้องกันได้จริงผ่านระบบควอนตัม ซึ่งเป็นแนวทางสำคัญในการวางแผนป้องกันระบบเครือข่ายเชิงรุกในอนาคต
\end{itemize}
